# Array Sum Calculator

A simple C++ program that reads two numbers, stores them in an array, calculates their sum, and displays the result.

## Features

* Accepts two numbers as input
* Stores values in an array
* Calculates the sum of the two numbers
* Displays the result in the console

## Example

```text
Input:
5
7

Output:
12
```

## Concepts Used

* Arrays
* User input (`cin`)
* Console output (`cout`)
* Basic arithmetic

## Built With

* C++
* Visual Studio Code
* MinGW GCC Compiler
